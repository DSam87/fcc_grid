grid-column-gap & grid-row-gap

Adding gaps between the cells of the grid

grid-gap

grid-gap is shorthand for grid-row-gap & grid-column-gap

Note

If grid-gap has one value, it makes gap for both column & row

justify-items & align-items

If you want all the item's content in your css grid to share the same alignment.


justify-items for horizontal alignment of content

align-items for horizontal alignment of content

Grid Items

These properties are used on the grid items themselves.

grid-column & grid-row

These properties are used for grid items themselves

To control the amount of columns/rows an item will consume, we use the grid line numbers.

Example

We define the start line and the end line.

In this example, item1 starts from the 1st line (the top) of the column line and extends to the 3rd line (closer to the).
It then expands itself along the row lines, starting from 1st line of the row (the left) and ending on the 4th row line (the right side).

.grid-item-1{
grid-column: 1/3
grid-row: 1/4
}

justify-self & align-self

The content of each item is located in a box which is called a cell of the grid.

The content inside the cell can be aligned with justify-self & align-self

justify-self aligns the cell's content horizontally

align-self aligns the cell's content verticaly

Grid-Template-Areas

Grid-Template-Areas (On Container)

These properties are used on the grid container.

grid-template-areas

You can group cells of your grid together into an AREA and give the area a custom name.

.grid-container{
"header header header"
"advert content content"
"footer footer footer";
}

Note that every word in the code represents a cell and every " " represents a row.

You can also use . to designaate an empty cell in the grid.

Grid-Area (On Items)

These properties are used on the grid item.

grid-area (with name)

Now with the grid-template-areas set and named, you can place an item in your custom area by referencing the name you gave it.

.grid-item-5{
grid-area: footer;
}

This lets the grid know that you want the item5 class to go in the area named header.

grid-area (with no name)

If your grid doesn't have grid-areas-template to reference, you can create an area on the fly for an item to be placed like this.

grid-area values:
horizontal line to start/ vertical line to start / horizontal line to end / vertical line to end

.grid-item-5{
grid-area: 1/1/2/4
}
